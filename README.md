## Features:
<li>Vulkan setup
<li>Resource Manager
<ul> 
<li>Meshes
<li>Textures
</ul>
<li>Multiple models in the scene
<li>Camera (with Free Cam mode)

</ul


## Controls:
- Free camera mode → ```Mouse Wheel Button```
- Camera's movements → ```WASD```, ```SPACE```, ```LEFT SHIFT```
